# HE Language Extension — Changelog

## 5.0.0 (2026)

Initial release.

- Full syntax highlighting for HE v5.0.0 (Pass 14)
- Tilde comment support `~ ... ~`
- String interpolation highlighting `"Hello, {name}!"`
- `#protected[N]` tag highlighting in orange
- Bracket matching and auto-close for `[`, `]`, `(`, `)`, `"`, `~`
- 35+ code snippets covering all major HE constructs
- **HE: Run File** command (Ctrl+F5)
- **HE: Check File** command (Ctrl+Shift+K) — type inference + protection tag report
- **HE: Build to .hex** — compile to self-contained bundle
- **HE: Open REPL** — interactive HE shell
- **HE: Run genesis.he** — run project entry point
- Check-on-save with Problems panel integration
- Right-click context menus in editor and Explorer
- Configurable `he` executable path
